# Future Continuous (Thì tương lai tiếp diễn)

## 1. Form (Cấu trúc)

### 1.1 Positive (Khằng định):
- `S + will + be + V-ing`

### 1.2 Negative (Phủ định):
- `S + will + not + be + V-ing`

### 1.3 Yes / No Questions (Câu hỏi Yes / No):
- `Will (not) + S + be + V-ing ?`
  - `Yes, S + will`
  - `No, S + will not`

### 1.4 Wh- Questions (Câu hỏi Wh):
- `Wh- + will (not) + S + be + V-ing ?`
