# Past Perfect (Thì quá khứ hoàn thành)

## 1. Form (Cấu trúc)

### 1.1 Positive (Khằng định):
- `S + had + V3`

### 1.2 Negative (Phủ định):
- `S + had + not + V3`

### 1.3 Yes / No Questions (Câu hỏi Yes / No):
- `Had (not) + S + V3 ?`
  - `Yes, S + had`
  - `No, S + hadn't`

### 1.4 Wh- Questions (Câu hỏi Wh):
- `Wh- + had (not) + S + V3 ?`
